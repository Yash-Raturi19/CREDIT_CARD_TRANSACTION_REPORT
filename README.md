# CREDIT_CARD_TRANSACTION_REPORT

📊 Credit Card Transaction Report (Power BI)

📌 Project Overview

This project presents a Credit Card Transaction Dashboard built using Microsoft Power BI. The report analyzes customer transactions, revenue, and spending patterns to generate meaningful business insights.

🎯 Objective

To analyze credit card usage trends
To identify revenue drivers and customer behavior
To track transaction volume and interest earned
To provide actionable insights for decision-making

📂 Dataset Description

The dataset includes the following key fields:

Card Category (Blue, Silver, Gold, Platinum)
Revenue
Transaction Amount
Interest Earned
Expense Type (Bills, Fuel, Food, etc.)
Transaction Mode (Swipe, Chip, Online)
Quarter & Weekly Data

📊 Dashboard Features

🔹 KPI Metrics

Total Revenue: 23.44M
Total Transactions: 19M
Total Interest Earned: 3.32M
Total Transaction Count: 278K

🔹 Visualizations

Quarterly Revenue & Transaction Trends
Revenue by Card Category
Revenue by Expense Type
Revenue by Transaction Mode
Interactive Filters (Quarter, Week Start Date)

📈 Key Insights

Blue card users contribute the highest share of revenue (~85%)
Revenue remains stable across all quarters (5.3M – 6.4M)
Bills and Fuel are the top spending categories
Swipe transactions dominate over chip and online payments
Online transactions are significantly lower, indicating growth opportunity

🛠 Tools & Technologies

Microsoft Power BI
Data Modeling
DAX (Data Analysis Expressions)
Data Cleaning & Transformation

💡 Business Recommendations

Focus on increasing online transactions through digital campaigns
Promote premium cards (Gold/Platinum) to diversify revenue
Offer targeted deals in high-spending categories (Bills, Fuel)
Improve customer engagement strategies for lower-tier card users

Conclusion

This dashboard provides a comprehensive view of credit card transactions,
helping businesses understand customer behavior, optimize strategies, and improve revenue generation.
